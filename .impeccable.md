## Design Context

### Users
Dispatchers (Jason y equipo EYAN) en escritorio durante jornada laboral.
Choferes en móvil/tablet en campo — pantalla de "Mis Casos" y reporte.
Admins para gestión general.

### Brand Personality
Operacional · Preciso · Decisivo

### Aesthetic Direction
Amber sobre navy-oscuro. Sala de control de despacho — no SaaS genérico.
- Sidebar: muy oscuro (oklch ~9%), separado visualmente del contenido
- Accent: amber cálido (oklch 72% / hue 54) — no el azul genérico de shadcn
- Tabla densa en lugar de grid de cards para la lista de casos
- Status como dot (●) + label, no badges pill
- Fuentes: Barlow Semi Condensed (headings/labels — sensación industrial) + Figtree (body) + JetBrains Mono (números de caso)

### Anti-patterns (Never)
- No border-left stripes en cards/rows
- No gradient text
- No glassmorphism decorativo
- No card-within-card nesting
- No blue-on-dark genérico con glow cyan/purple

### Design Principles
1. Densidad con claridad — más información visible sin saturar
2. Jerarquía tipográfica fuerte — Barlow condensed para labels, mono para IDs
3. Color funcional — amber solo para acciones/primario, dots para estados
4. Operacional primero — diseñado para uso rápido, no estético
5. Dark with purpose — oscuro porque despacho/jornada laboral, no por moda
